## Installation

1. Clone the repository:
```bash
  git clone https://github.com/yourusername/job-landing-page.git
```

2. Navigate to the project directory:
```bash
  cd job-landing-page
```

3. Install dependencies:
```bash
  npm install
```

4. Run the development server:
```bash
  npm run dev
```
